📋 Вариант 1: Локальный dev-сервер (рекомендуется)

1. Откройте новый PowerShell (не в VS Code, чтобы избежать Git-конфликтов)
2. Установите Node.js (если ещё нет): 
winget install OpenJS.NodeJS.LTS
После установки перезапустите PowerShell.
3. Перейдите в папку проекта: 
cd "C:\Users\manso\Desktop\разработка\your-ai-companion-main"
4. Установите зависимости: 
npm install
5. Запустите dev-сервер: 
npm run dev
✅ Сайт будет доступен:
http://localhost:8080

📦 Вариант 2: GitHub Pages (уже настроено)

1. Допушьте проект в GitHub (если ещё не запушен): 
git config core.editor "notepad"
git merge --abort
git rebase --abort
git push -u origin main --force
2. В GitHub → Settings → Pages:
  • Source: GitHub Actions
3. В GitHub → Settings → Secrets → Actions:
  • VITE_SUPABASE_URL
  • VITE_SUPABASE_PUBLISHABLE_KEY
✅ Сайт будет доступен:
https://kmansoni.github.io/ECOMANSONI/
