# Contract Examples (Golden Traces)

Цель: набор валидных примеров JSON, которые должны проходить соответствующие JSON Schema.

Правила:
- Любое изменение схемы должно сопровождаться обновлением примеров.
- Примеры должны отражать реальные поля/значения (без «магических» новых фич).

Быстрая ручная проверка:
- Открой схему в `docs/contracts/schemas/` и пример в `docs/contracts/examples/`.
- Сверь required/enum/format и `$ref` зависимости.

Автоматическая проверка (опционально):
- Можно использовать `ajv-cli` (Node) чтобы валидировать примеры против схем.
