**Цель:** 100 000 установок приложения за 7 дней

**Day 0 (Мягкий запуск — Закрытое бета):**
- Founders программа: 500 продавцов + 2 000 покупателей из waitlist
- Команда мониторит каждый заказ вручную
- Hotline: прямой телефон CEO для продавцов в первую неделю

**Day 3 (Широкий запуск):**

- VK Ads + Telegram Ads запуск с бюджетом ₽3 млн/нед

- Акция: "Промокод СТАРТ500: ₽500 на первый заказ" — массовая рассылка

**Day 7:**
- Пресс-конференция с ключевыми продавцами-основателями
- NPS первой волны покупателей → немедленные исправления по фидбеку

#### Post-Launch (Месяц 1–6)

**Цели:**
- 1M зарегистрированных покупателей
- 10 000 активных продавцов
- GMV ₽2 млрд к 6-му месяцу

## 4. Контент-Маркетинг и SEO

### 4.1 SEO Стратегия

**Технический SEO:**
```
Приоритеты технического SEO:
1. Core Web Vitals: все страницы LCP < 2.5s, CLS < 0.1, INP < 200ms
2. Structured Data (Schema.org):
   - Product schema (всех товаров)
   - BreadcrumbList (навигации)
   - FAQPage (страниц помощи)
   - Review (агрегированные рейтинги)
3. Canonical URLs: правильные canonical для фильтрованных страниц
4. XML Sitemap: автогенерация, обновление при новых товарах
5. Robots.txt: правильное закрытие search?q=, filter params
6. Hreflang: ru-RU, en-US для международных страниц
7. Page Speed: ISR на всех category/product страницах
```

## 6. CRM и Lifecycle Marketing

### 6.1 Сегментация пользователей (RFM)

```
R (Recency): дней с последней покупки
F (Frequency): покупок за 90 дней
M (Monetary): сумма покупок за 90 дней

Сегменты:
┌─────────────────────────────────────────────────────────────┐
│  Champions        │ R<7 d, F>3, M>₽10K │ Best customers     │
│  Loyal            │ R<30d, F>2, M>₽5K  │ Regular buyers     │
│  Recent           │ R<14d, F=1, M<₽5K  │ New customers      │
│  At Risk          │ R 30-90d, was loyal  │ Need reactivation  │
│  Lost             │ R>90d               │ Winback campaign   │
│  Promising        │ R<30d, F=1-2       │ Nurture            │
│  Sleeping         │ R 30-90d, low F    │ Light nudge        │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 Триггерные цепочки по сегментам

**New User (первые 7 дней):**
```
Day 0: Приветственное письмо
  - "Привет, [Имя]! Вот ваш промокод СТАРТ500"
  - Топ-5 категорий на основе данных регистрации
  - CTA: "Начать покупки"

Day 2 (если не купил): Push + Email
  - "Ваш промокод истекает через 5 дней"
  - "Посмотрите, что популярно сейчас"

Day 5 (если не купил): Push
  - "Последний шанс: ₽500 скоро сгорят"
  - Персонализированная подборка

Day 7 (если не купил): Email
  - "Мы подобрали специально для вас..."
  - Социальное доказательство: "2 млн человек уже купили на NEXUS"
```

**Cart Abandonment:**
```
T+30 мин: Push (если включён)
  - "Вы забыли что-то в корзине 🛒"
  - Превью товаров с фото
  - CTA: "Завершить покупку"

T+2 часа: Email
  - Subject: "[Имя], ваша корзина скучает по вам"
  - Product images + цены
  - "Цена может измениться"

T+24 часа: Email (если дорогой товар > ₽5 000)
  - "Специально для вас: +200 Stars к этому заказу"
  - Limited time: следующие 24 часа

T+72 часа: Последнее письмо
  - "Последний шанс сохранить корзину"
  - Рекомендации альтернатив
```

**At Risk (30+ дней без покупки, was loyal):**
```
Email 1 (День 31):
  Subject: "[Имя], мы скучаем! Что-то новенькое для вас"
  - Персонализированные новинки в любимых категориях
  - "Посмотрите, что появилось"

Email 2 (День 45):
  Subject: "Специальное предложение только для вас"
  - Уникальный промокод 10% скидки
  - Срок: 7 дней

Email 3 (День 52):
  Subject: "Ваши Stars сгорят через 30 дней"
  - Показать баланс Stars
  - "Не дайте им пропасть"

Push (День 60):
  - "Возвращайтесь! Ваши ₽[баланс Stars] ждут вас"
```

**Win-back (90+ дней без активности):**
```
Email: "Много всего изменилось за последние 3 месяца"
  - Новые категории / продавцы
  - Улучшения платформы
  - Специальный "Возвращайся!" промокод 15%
  - "Или удалите аккаунт — мы уважаем ваш выбор" (снижает unsubscribe)

Если нет реакции за 60 дней после winback → переход в "Churned"
→ Email frequency снижается до 1/мес (compliance с unsubscribe предпочтениями)
```

### 6.3 Push Stратегия

**Ограничения push-количества (Anti-Spam политика NEXUS):**
- Максимум 2 маркетинговых push в день
- Никаких push между 22:00 и 8:00 (local time)
- Пользователь настраивает свои предпочтения
- Smart Throttling: если CTR push упал < 1% → снизить частоту автоматически

**Персонализированные push (на основе ML-модели):**
- "Цена снизилась на [товар из избранного]"
- "Начался стрим вашего любимого продавца [Имя]"
- "Ваш заказ #NX-12345 доставлен"
- "Акция в [любимой категории]: -20% сегодня"

## 7. Вирусные Механики и Реферальный Рост

### 7.1 Реферальная Программа

```
Механика "Позови друга":
- Реферер получает: 500 Stars (= ₽5) после первой покупки друга
- Новый пользователь получает: ₽500 промокод
- Ссылка работает 30 дней
- Лимит: 100 успешных рефералов/год на аккаунт (анти-мошенничество)

KPI реферальной программы:
- Viral Coefficient K = (# рефералов от 1 пользователя за месяц) × (конверсия реферала)
- Target: K = 0.4 (не вирусный, но значимый дополнительный рост)
- При K=0.4: каждые 1 000 привлечённых → +667 через рефералы → итого 1 667
```

### 7.2 Social Commerce (встроенные вирусные петли)

**Механики шеринга:**
- Каждый товар: кнопка "Поделиться" → генерирует OG-card с фото + ценой
- "Лук дня": покупатель собирает образ → публикует → другие покупают через него → покупатель получает Stars
- Live стрим: зрители могут пригласить друзей в стрим → реферальный бонус
- Wish List шеринг: "Мой вишлист на день рождения" → ссылка → другие покупают
