# Generated avatars (SVG)

These are lightweight, generic (non-real-person) SVG avatars intended for Profile / Edit Profile.

Paths:
- `/avatars/generated/president-1.svg` .. `president-4.svg`
- `/avatars/generated/king-1.svg` .. `king-4.svg`
- `/avatars/generated/sultan-1.svg` .. `sultan-4.svg`

Usage example:

```tsx
<img src="/avatars/generated/president-1.svg" alt="Avatar" />
```
