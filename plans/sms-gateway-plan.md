# План: Собственный SMS-шлюз на базе GSM-модема

## Текущая ситуация
- Проект использует **SMS.ru API** для OTP
- Зависимость от стороннего сервиса
- Файл: `supabase/functions/send-sms-otp/index.ts`

## Цель
Создать собственный SMS-шлюз без зависимости от операторов и third-party сервисов.

## Оборудование

### Рекомендуемые модемы
| Модем | Цена | Производительность | Примечание |
|-------|------|-------------------|------------|
| Huawei E1550 | ~500-1000₽ | 10-30 SMS/час | Самый популярный |
| Huawei E173 | ~800-1500₽ | 10-50 SMS/час | Более стабильный |
| Huawei E3372 | ~1500-2500₽ | 20-100 SMS/час | LTE поддержка |
| SIM800L | ~300-500₽ | 5-20 SMS/час | Arduino-совместимый |

### Требования к SIM
- Тариф с бесплатными SMS (МТС "Супер", Билайн, Мегафон)
- Статический IP не требуется
- Достаточно обычного тарифного плана

## AT-команды для отправки SMS

```
# Инициализация модема
AT
AT+CMGF=0          # PDU режим
AT+CMGS=<length>  # Отправка SMS

# Чтение SMS
AT+CMGL="ALL"      # Список всех SMS
AT+CMGR=<index>    # Чтение конкретного SMS

# Настройки
AT+CSCA=<smsc>     # SMS Center номер
AT+CSCS="GSM"      # Кодировка
```

## Ограничения и решения

| Проблема | Решение |
|----------|---------|
| Модем занят | Пул из нескольких модемов |
| Оператор блокирует | Ротация SIM-карт |
| Очередь переполнена | Приоритезация (OTP важнее маркетинга) |
| Модем отключился | Auto-reconnect + health checks |
