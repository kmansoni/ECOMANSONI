# Archive Agents Policy

Этот каталог содержит **legacy Claude agent definitions**.

Правила:

- архивные агенты не считаются активным execution layer
- канонический entrypoint проекта: `mansoni`
- прямой runtime mode: `ruflo`
- архивные файлы нельзя рассматривать как второй источник истины

Канонические активные агенты находятся в:

- `.claude/agents/mansoni.agent.md`
- `.claude/agents/mansoni-core.agent.md`
- `.claude/agents/ruflo.agent.md`
